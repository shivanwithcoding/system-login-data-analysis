# System Login Data Analysis Project

## 📌 Project Overview
This project analyzes simulated system login data using Python.  
It focuses on exploring user login patterns, success/failure rates, and detecting unusual activity using data analysis techniques.

## 🛠️ Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook

## 📂 Dataset
The dataset contains login records with the following columns:
- timestamp
- user_id
- ip_address
- country
- success
- os

## 📊 Analysis Performed
- Data cleaning and preparation
- Analysis of login success vs failure
- User activity patterns
- Visualization using graphs

## 📈 Results
The analysis helps identify:
- Most active users
- Login success rates
- Suspicious or failed login attempts

## ▶️ How to Run
1. Open Jupyter Notebook
2. Run all cells from top to bottom
3. Ensure required libraries are installed:

pip install pandas numpy matplotlib
